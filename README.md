# Stock Data Extraction Using Python

This repository contains two Jupyter notebooks demonstrating different methods to extract stock market data using Python. The project focuses on understanding real-world data collection techniques used in financial analysis and data science.

## Files Included

- `extracting_stock_data_using_python_library.ipynb`  
  Extracts historical stock data using a Python financial data library.

- `extracting_stock_data_using_web_scrapping.ipynb`  
  Extracts stock data from financial websites using web scraping techniques.

## Key Concepts Covered

- Stock market data extraction  
- API-based data collection  
- Web scraping with BeautifulSoup  
- Data handling using Pandas  

## Technologies Used

- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Requests  
- BeautifulSoup  
- yfinance  

## How to Run

1. Clone the repository  
2. Open the notebooks using Jupyter Notebook  
3. Run the cells sequentially  

## Learning Outcome

- Gained practical experience in collecting stock market data  
- Understood the advantages and limitations of APIs vs web scraping  
- Built a foundation for financial data analysis projects
